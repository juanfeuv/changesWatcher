# App assesment with spring boot

## System requirements
* Using java 17
* Using MongoDB 6.0
* Using mvn 3.8.7(Apache Maven 3.8.7)

## Get started

### Install dependencies
`mvn clean install`

### Run Mongodb localhost:27017
`mongod`

### Run the app
`mvn spring-boot:run`

## Swagger docs
You should go to the path in the browser: http://localhost:8080/swagger-ui.html
